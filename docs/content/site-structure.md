## Technical Details (Original Site)

- **CMS:** WordPress 5.3.20
- **Theme:** Custom theme (miciicampioni-v1.0)
- **CSS Framework:** Bootstrap
- **Slider:** bxSlider jQuery plugin
- **Lightbox:** Responsive Lightbox Nivo
- **Animation:** Custom sea creature animations (whale, crab, seahorse, tortoise, fish)
- **SEO:** Yoast WordPress SEO plugin v1.4.25
- **Analytics:** Google Analytics (UA-1736378-5)
- **Responsive:** Yes (3 navigation states: left-right, center, dropdown mobile)
